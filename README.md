Zadanie 1 – Losowe przestawienie liter w pliku tekstowym

Opis:
Aplikacja umożliwia wgranie pliku tekstowego i przetwarza jego zawartość w taki sposób, że w każdym wyrazie zostają zachowane pierwsza i ostatnia litera, a pozostałe litery są losowo mieszane. Program uwzględnia:

interpunkcję,

wielkie i małe litery,

polskie znaki,

tekst wieloliniowy.

Funkcjonalności:

Upload pliku .txt.

Wyświetlenie przetworzonego tekstu w przeglądarce.

Zachowanie struktury wierszy i znaków interpunkcyjnych.


Zadanie 2 – Walidacja numeru PESEL

Opis:
Aplikacja umożliwia wprowadzenie numeru PESEL i sprawdza jego poprawność zgodnie z oficjalnym algorytmem. Walidacja obejmuje:

długość numeru (11 cyfr),

datę urodzenia zakodowaną w numerze PESEL,

sumę kontrolną.

Funkcjonalności:

Formularz do wpisania numeru PESEL.

Informacja zwrotna, czy numer jest poprawny.

Walidacja w czasie rzeczywistym.

Testy jednostkowe:

Testy sprawdzające numery niepoprawne (krótsze, dłuższe, błędna suma kontrolna).

Test poprawnego numeru PESEL.



Zadanie 3 – Obsługa API GoREST

Opis:
Aplikacja integruje się z publicznym API GoREST (https://gorest.co.in/) i pozwala na przeglądanie oraz edycję użytkowników.

Funkcjonalności:

Wyświetlenie listy użytkowników pobranych z API.

Wyszukiwanie użytkowników po imieniu lub emailu.

Edycja istniejących wpisów użytkowników (np. imię, email).

Aktualizacja widoku po edycji.
